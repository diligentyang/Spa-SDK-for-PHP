Spa\App

实例介绍

Spa\App


    new Spa\App({$uid}, {$appid}, {$appkey})
初始化要提供用户的 uid appid appkey 一般由 Spa\Spa 类初始化，可以直接通过 Spa\Spa 的 getApp() 方法获取实例 例： 

    $conf = array(
        'uid'               => '{uid}',
        'appid'             => '{appid}',
        'appkey'            => '{appkey}',
        'http_client_type'  => 'curl',
        'version'           => 'v3'
    );

    $spa = new Spa\Spa($conf);

    $spaApp = $spa->getApp();
